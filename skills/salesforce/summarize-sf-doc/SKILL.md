Summarize the referenced document from Salesforce.

In the summarization, when there are abbreviations put what the abbreviation stands for in square brackets. For example, "FKP [Falcon Kubernetes Platform]". Write the definition every time the abbreviation is used.

Whenever you put the definition of the abbreviation in the summarization highlight it by italicizing the definition in the square brackets.

Use the decode-sf skill to decode abbreviations.

Do not decode these abbreviations:
- Ajna
- Airflow
- AMI
- API
- AWS
- BDIP
- DAG
- DQ
- EBS
- EC2
- EMR
- ETL
- GCP
- GenAI
